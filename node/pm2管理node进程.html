# pm2定时任务

$ pm2 start <app_name|namespace|id|'all'|json_conf>
$ pm2 stop     <app_name|namespace|id|'all'|json_conf>
$ pm2 restart  <app_name|namespace|id|'all'|json_conf>
$ pm2 delete   <app_name|namespace|id|'all'|json_conf>

# pm2 start with name
pm2 start app.js --name "my-pm2-app"

# 问题

1.crontab配置pm2时报错pm2 node no such file or directory

原因是bash找不到node bin程序

解决方案：ln -s /usr/bin/nodejs /usr/bin/node

如果node是用nvm安装的话，就是 ln -s $NVM_DIR/versions/node/v15.14.0/bin/node /usr/bin/node

# 事件冒泡

JS阻止冒泡和取消默认事件(默认行为) (opens new window)

js冒泡和捕获是事件的两种行为，使用event.stopPropagation()起到阻止捕获和冒泡阶段中当前事件的进一步传播。使用event.preventDefault()可以取消默认事件。对于冒泡和捕获的优先顺序请看之前文章：JavaScript捕获和冒泡探讨 (opens new window)

return false也可以用来阻止默认行为，只不过得用在onclick内嵌中

<a href="https://www.baidu.com/" onclick="{console.log('onclick'); return false;}">
  123
</a>

# 重绘和重排

# 重排

定义: 重排也称为 回流，是指当页面的布局或元素的几何属性（尺寸、位置、形状等）发生变化时，浏览器需要重新计算页面布局。

重排必然导致重绘，因为页面布局改变后，元素的视觉外观需要更新。

重绘不一定导致重排，例如改变颜色只会影响视觉层，不会影响布局。

transform 的变化 不会触发重排（Reflow），只会触发 重绘（Repaint）。

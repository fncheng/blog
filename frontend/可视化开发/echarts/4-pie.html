# 2. 设置legend图例和文字的位置

legend.align (opens new window)

图例标记和文本的对齐。默认自动，根据组件的位置和 orient 决定，当组件的 left (opens new window) 值为 'right' 以及纵向布局（orient (opens new window) 为 'vertical'）的时候为右对齐，即为 'right'。

# 4. 移动饼图

series-pie.center (opens new window)

饼图的中心（圆心）坐标，数组的第一项是横坐标，第二项是纵坐标。

支持设置成百分比，设置成百分比时第一项是相对于容器宽度，第二项是相对于容器高度。

  title: {
    text: '',

  color: ['#49B6FF', '#FF7B8F', '#FB8F67', '#48D4BB'],

  legend: {

# 7.饼图数据过小时百分比会显示为0

series.pie.label.formatter

打印param

可以看出value=22，而percent=0

解决办法：

formatter(param) {
  if (Number(param.value) && Number(param.percent)) {
    return `${param.percent}%`
  } else return ''
}

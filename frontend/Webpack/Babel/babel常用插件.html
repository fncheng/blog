# @babel/polyfill

这个包已被弃用，取而代之的是**core-js (opens new window) and regenerator-runtime (opens new window).**

# core-js

JavaScript 的模块化标准库。包括ECMAScript 到 2021 年的 polyfills (opens new window)：promises (opens new window)、symbols (opens new window)、collections (opens new window)、iterators、typed arrays (opens new window)、许多其他特性、ECMAScript 提案 (opens new window)、一些跨平台的 WHATWG/W3C 特性和提案， (opens new window)如URL (opens new window). 您可以仅加载所需的功能或在没有全局命名空间污染的情况下使用它。

### tree-shaking

https://juejin.cn/post/7004297344300777502

Tree-shaking只支持ESM的引入方式，不支持 Common JS 的引入方式。

- ESM: export + import
- Common JS: module.exports + require

```js
import { add } from 'lodash'	// 不支持
import { add } from 'lodash-es' // 支持
```

原因是lodash是module.exports 导出的

## 表格元素

[html表格入门](https://developer.mozilla.org/zh-CN/docs/Learn/HTML/Tables/Basics)

- tr	table row的缩写

- th   table head 的缩写

- td   table data的缩写，表格数据单元格

## 表格边框

```css
table {
      border: 1px solid salmon;
      border-collapse: collapse;
    }
    th,td {
      border:  1px solid #000;
      border-collapse: collapse;
    }
```

### 合并边框

border-collapse

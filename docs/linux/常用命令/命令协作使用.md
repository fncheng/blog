## ls grep rm

使用 xargs 给其他命令传递参数的一个过滤器

```sh
ls | grep 'Con' | xargs cat
# ll不支持, 因为ll不是单纯的搜索文件
# ll=ls -lh
```

- -t 表示先打印命令，然后再执行。
- -i 或者是-I，这得看linux支持了，将xargs的每项名称，一般是一行一行赋值给 {}，可以用 {} 代替。

```sh
$ ls | grep 'Con' | xargs -t ls
ls Config.json
Config.json
```

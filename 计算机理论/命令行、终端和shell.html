# 命令行界面 (CLI)、终端 (Terminal)、Shell、TTY的区别 (opens new window)

http://lizengzhang.com/88/

  • 命令行界面 (CLI) = 使用文本命令进行交互的用户界面
  • 终端 (Terminal) = TTY = 文本输入/输出环境
  • 控制台 (Console) = 一种特殊的终端
  • Shell = 命令行解释器，执行用户输入的命令并返回结果

# 输入重定向和输出重定向

https://www.runoob.com/linux/linux-shell-io-redirections.html

输出重定向就是将原本的输出（比如从终端输出）变到从其他地方输出（比如输出到文件）

输入重定向则相反，改为从其他地方（比如文件）获取输入，而不是从终端获取输入。

>和>>是输出重定向

<是输入重定向

# firewalldе’Ңfirewall-cmd

firewall-cmd е’Ң firewalld жҳҜ CentOS/RHEL зі»з»ҹдёӯз”ЁдәҺй…ҚзҪ®е’Ңз®ЎзҗҶйҳІзҒ«еўҷзҡ„дёӨдёӘе…іиҒ”е·Ҙе…·гҖӮ

firewalld жҳҜдёҖдёӘеҠЁжҖҒзҡ„йҳІзҒ«еўҷз®ЎзҗҶеҷЁпјҢз”ЁдәҺй…ҚзҪ® iptables и§„еҲҷгҖӮе®ғжҸҗдҫӣдәҶдёҖдёӘ D-Bus жҺҘеҸЈе’ҢдёҖдёӘе‘Ҫд»ӨиЎҢжҺҘеҸЈпјҢеҸҜд»ҘйҖҡиҝҮиҝҷдёӨдёӘжҺҘеҸЈиҝӣиЎҢйҳІзҒ«еўҷи§„еҲҷзҡ„й…ҚзҪ®е’Ңз®ЎзҗҶгҖӮfirewalld е…Ғи®ёз”ЁжҲ·еңЁиҝҗиЎҢж—¶еҠЁжҖҒең°жӣҙж”№йҳІзҒ«еўҷи§„еҲҷпјҢиҖҢж— йңҖйҮҚеҗҜйҳІзҒ«еўҷжңҚеҠЎгҖӮ

firewall-cmd жҳҜз”ЁдәҺдёҺ firewalld дәӨдә’зҡ„е‘Ҫд»ӨиЎҢе·Ҙе…·гҖӮе®ғжҸҗдҫӣдәҶдёҖз§Қз®ҖеҢ–зҡ„ж–№ејҸжқҘй…ҚзҪ® firewalldпјҢе№¶жү§иЎҢйҳІзҒ«еўҷж“ҚдҪңгҖӮfirewall-cmd е‘Ҫд»ӨйҖҡиҝҮи°ғз”Ё firewalld зҡ„ D-Bus жҺҘеҸЈжқҘе®һзҺ°йҳІзҒ«еўҷзҡ„й…ҚзҪ®гҖӮ

дё»иҰҒе‘Ҫд»Өе’Ңз”Ёжі•пјҡ

  • --add-serviceпјҡ ж·»еҠ дёҖдёӘжңҚеҠЎгҖӮ
  • --add-portпјҡ ж·»еҠ дёҖдёӘз«ҜеҸЈгҖӮ
  • --list-allпјҡ еҲ—еҮәеҪ“еүҚйҳІзҒ«еўҷи§„еҲҷзҡ„иҜҰз»ҶдҝЎжҒҜгҖӮ
  • --reloadпјҡ йҮҚж–°еҠ иҪҪйҳІзҒ«еўҷи§„еҲҷгҖӮ
firewall-cmd --zone=public --add-port=8888/tcp --permanent	#ејҖеҗҜз«ҜеҸЈ
firewall-cmd --list-ports	#жҹҘзңӢејҖж”ҫзҡ„з«ҜеҸЈ
firewall-cmd --query-port=8888/tcp	#жҹҘзңӢз«ҜеҸЈжҳҜеҗҰејҖеҗҜ
lsof -i:8888	#жҹҘзңӢз«ҜеҸЈеҚ з”Ё
netstat -tunlp |grep 8888	#жҹҘзңӢжҢҮе®ҡз«ҜеҸЈзҡ„иҝӣзЁӢжғ…еҶө
